# Google Map 高分店家指南系統

這是一套完整的前後端整合服務，提供「**現代化直覺 UI 介面**」與「**真實 Google Map API 串接**」，能幫助你快速發掘各地區的高分美食與店家，以及篩選優質旅館/飯店。不僅支援跨平台響應式設計，更具備強大的時段感知與多語系功能。

## 🚀 核心專案功能

1. **現代化響應式 UI (RWD)**：使用 Tailwind CSS 打造高質感介面，並完整支援手機、平板與電腦版面。按鈕圖示化（SVG 搜尋按鈕）讓畫面更簡潔。
2. **🍜🏨 雙模式搜尋（美食 / 旅館）**：
   - **美食模式**：搜尋地區餐廳、小吃、咖啡廳、甜點等美食店家。
   - **旅館模式**：搜尋地區飯店、旅館、民宿、酒店，並**自動過濾 3.5 星以上**的優質住宿。
   - 透過 Tab 切換按鈕一鍵切換，搜尋按鈕、隨機抽籤按鈕及表格配色皆隨模式變換主題色（藍色 → 紫色）。
3. **智慧擴充搜尋與去重複**：突破 Google Maps API 單次最高 60 筆的限制。透過智慧化關鍵字展開（如自動拆分餐廳/小吃/咖啡廳、飯店/旅館/民宿/酒店），一次最高可網羅 240 筆不重複的當地結果。
4. **超高速並行查詢**：利用 Python `asyncio` 非同步多執行緒架構，讓擴展後的龐大搜尋任務平行處理，將原本需要數秒的等待時間大幅縮短。
5. **🎲 時段感知隨機抽籤**：
   - 解決「今天吃什麼」的選擇困難！點擊按鈕，系統會自動在搜尋結果中為你隨機抽籤。
   - **時段感知功能**（美食模式）：系統會依據你當下的時間，自動過濾出符合該時段的餐廳。
   - **旅館模式**：隨機推薦一間高評分住宿，並顯示價位等級與官方網站連結。
6. **🌐 多語系即時切換 (i18n)**：支援**繁體中文**、**簡體中文**與 **English**。切換語系不僅會即時更新畫面文字（無需重整），更會同步改變 Google Maps API 回傳的地點語言！
7. **一鍵導航地圖**：前端渲染的表格與抽籤視窗中，店家地址皆附有 Google Maps 專屬連結，點擊即可開新分頁精準定位。

## 📂 目錄架構

```text
SearchFoodMap/
├── frontend/                 # 前端專案資料夾
│   ├── index.html            # UI 介面 (支援 RWD、語系選單、雙模式切換)
│   └── app.js                # 前端互動邏輯、翻譯字典、分頁與抽籤控制
├── backend/                  # 後端專案資料夾
│   ├── main.py               # FastAPI 伺服器與非同步核心邏輯 (美食 + 旅館)
│   └── requirements.txt      # Python 套件依賴
└── Dockerfile                # 容器化部署設定檔
```

## 📥 安裝與啟動

我們採用了 FastAPI 作為後端核心，並直接讓後端伺服器負責提供靜態的前端檔案，確保您能「一鍵順暢啟動」。

請開啟終端機並執行以下指令：

```powershell
# 1. 進入專案目錄
cd SearchFoodMap

# 2. 安裝後端所需套件
pip install -r backend/requirements.txt

# 3. 【重要】設定您的 Google Maps API 金鑰 (請替換成真實的金鑰)
$env:GOOGLE_MAPS_API_KEY="您的_API_KEY"

# 4. 啟動伺服器
python backend/main.py
```
> **提示**：如果在開發過程中需要熱重載 (Hot Reload)，可以使用指令 `uvicorn backend.main:app --reload`。

## 🎮 如何使用 UI

伺服器啟動後，請打開您的瀏覽器，前往：**[http://localhost:8080/](http://localhost:8080/)**

1. **切換搜尋模式**：在標題下方點選「🍜 美食餐廳」或「🏨 旅館飯店」Tab 切換搜尋模式。
2. **搜尋餐廳/旅館**：輸入地區名稱（例如：`台南市中西區` 或 `台北市信義區`）並點擊放大鏡按鈕。
3. **切換語系**：在畫面右上角的下拉選單，可以隨時無縫切換多種語言介面。
4. **隨機抽推薦**：搜尋完成後，點擊「🎲 隨機抽籤」（或「🎲 隨機推薦住宿」），讓系統為你推薦！
